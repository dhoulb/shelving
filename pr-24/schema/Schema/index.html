SchemaOptions

type SchemaOptions = { /** String for one of this thing, e.g. `product` or `item` or `sheep` */ readonly one?: string; /** String for several of this thing, e.g. `products` or `items` or `sheep` (defaults to `one` + "s") */ readonly many?: string; /** Title of the schema, e.g. for using as the title of a corresponding field. */ readonly title?: string | undefined; /** Description of the schema, e.g. for using as a description in a corresponding field. */ readonly description?: string | undefined; /** Placeholder of the schema, e.g. for using as a placeholder in a corresponding field. */ readonly placeholder?: string | undefined; /** Default value for the schema if `validate()` is called with an `undefined` value. */ readonly value?: unknown; }

Options allowed by a Schema instance.

Schema

class
class Schema

Schema is an object instance with a validate() method.

  • Type T represents the type of value validate() returns.
  • validate() returns Invalid if value was not valid.
