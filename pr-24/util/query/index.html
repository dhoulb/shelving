QueryFilter

type
type QueryFilter = | { key: DataPath; operator: "is"; value: unknown } | { key: DataPath; operator: "not"; value: unknown } | { key: DataPath; operator: "in"; value: ImmutableArray } | { key: DataPath; operator: "out"; value: ImmutableArray } | { key: DataPath; operator: "contains"; value: unknown } | { key: DataPath; operator: "lt"; value: unknown } | { key: DataPath; operator: "lte"; value: unknown } | { key: DataPath; operator: "gt"; value: unknown } | { key: DataPath; operator: "gte"; value: unknown }

A single filter that can be applied to a list of data objects.

QueryOrder

type
type QueryOrder = { key: DataPath; direction: "asc" | "desc"; }

A single sort order that can be applied to a list of data objects.
