({ path = "/", tree }: TreePageProps) => ReactNode

Resolve a URL path to a tree element and render it. - Uses `resolveElementPath()` to walk the tree matching each path segment to a descendant's `key`. - `/` renders `tree` itself; deeper paths render the matching descendant. - Delegates rendering to the component registered in the element mapper. - Throws `NotFoundError` if no element matches at any level.

Parameters

{ path = "/", tree }: TreePageProps

Returns

ReactNode
