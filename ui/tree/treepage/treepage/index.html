({ path = "/", elements }: TreePageProps) => ReactNode

Resolve a URL path to a tree element and render it. - Uses `resolveElement()` to walk the tree matching each path segment to an element's `key`. - Delegates rendering to the component registered in the element mapper. - Throws `NotFoundError` if no element matches at any level.
