TreeApp

({ tree, routes = {}, children = <RouterOutput />, ...appProps }: TreeAppProps) => ReactElement

Top-level app component for a tree-based documentation site. - Wraps `<App>` with routing, error catching, and a sidebar layout. - The sidebar shows a `<TreeMenu>` of top-level elements. - `/` renders the root via `<TreePage>`; `/**` catches every deeper path and feeds the full sub-path into `<TreePage>`. - Element rendering uses the default mappings on `<TreePage>`, `<TreeMenu>`, `<TreeCards>`. Override by wrapping with `<TreePageMapping>`, `<TreeMenuMapping>`, or `<TreeCardMapping>`.
