Combo box with query input. - Show an input based on a `schema` of type `I` - Values from the input are piped to an `onQuery()` callback. - Show a `<Popover>` that has a radio list of items to allow a final option of type `O` to be picked. - Errors / loading state / empty state are handled automatically.

Parameters

{ empty = "No results", value, onValue, onQuery, formatter = formatValue, ...props }: QueryInputProps<I, O>

ReactElement
