ClickableCallback

(event: MouseEvent<HTMLButtonElement>) => ReactNode | void | PromiseLike<ReactNode | void>

Handler for a clickable `onClick` event. - Returned value (if defined) is notified to the user using `notifySuccess()` - Thrown value is notified to the user using `notifyError()`

ClickableProps

Props for a thing that can be clicked, either has a string `href` link or an `onClick` callback handler.

getClickable

(props: ClickableProps, className?: string) => ReactElement

Return either a `<button>` or an `<a href="">` based on whether an `onClick` or `href` prop is provided.
