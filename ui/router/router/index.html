RouterProps

Router

({ routes, children = <RouterOutput /> }: RouterProps) => ReactElement

Provides Router based on the browser URL and renders the active route. - Creates a new `RouterStore` - Exposes it via `requireRouter()` to descendants. - Renders the route matched for the current path. - Intercepts same-origin anchor clicks (excluding anchors with a `download` attribute) to perform client-side routing. - Synchronizes store state with browser back/forward (`popstate`) events. Integrated with meta via the current `Meta` - Set `url` in a wrapper component to set the initial URL for this router, e.g. `<App url="/a/b/c">` - Set `base` , e.g. `<App base="https://p.com/subdir" />`

RouterIsolateProps

RouterIsolate

({ children }: RouterIsolateProps) => ReactElement

Isolate a set of children so they remount on Router. - Use this when you want to ensure an entire React tree is remounted when the Router route changes.

RouterOutput

() => ReactElement | null

Show the content active for the current router route.
