(initial: T, delay = 500) => [T, (v: T) => void]

Set a state using a "debounce" delay — this ensures that the state is only updated after the specified delay has passed since the last update. - Useful for input fields where you want to wait until the user has stopped typing before performing an action (e.g., API call).

Parameters

initial: T
delay (optional)

Returns

[T, (v: T) => void]
