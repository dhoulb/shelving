useScrollIntersect

(onEnter?: Nullish<Callback>, onLeave?: Nullish<Callback>) => RefObject<T | null>

Fires a callback when an element enters the visible scrolling area on the client. - Checks that is the last element in its parent. - Finds the closest parent above this that scrolls (e.g. `.page`, customisable with `selector` param), then fires the callbacks appropriately.
