notify

(message: ReactNode, status?: Status | undefined, el: EventTarget = window) => void

Notify the user with a message by dispatching a notice event. - This is how e.g. `<Button>` and `<FormNotify>` components send notices to the `<Notices>` list of global notices.

notifySuccess

(message: ReactNode, el?: EventTarget) => void

Notify the user with a success message by dispatching a notice event.

notifyError

(message: ReactNode, el?: EventTarget) => void

Notify the user with a success message by dispatching a notice event.

notifyThrown

(thrown: unknown, el?: EventTarget) => void

Look at a thrown value, extract a viable message from it, and dispatch a notice event..

subscribeNotices

(callback: (message: ReactNode, status?: Status | undefined) => void, el: EventTarget = window) => () => void

Subscribe to notice events on the window and call a callback when they happen.

NoticeCallback

(...args: A) => PromiseLike<ReactNode | undefined | void> | ReactNode | undefined | void

Callback that can return or throw a string, which will trigger a success or error notice accordingly.
