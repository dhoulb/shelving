MetaTags

ImmutableDictionary<Nullish<string | boolean>>

Set of named meta `<meta />` tags in `{ rel: href }` format.

MetaLinks

ImmutableDictionary<Nullish<string>>

Set of named meta `<link />` tags in `{ rel: href }` format.

MetaAssets

ImmutableArray<Nullish<string>>

Set of linked assets in `(href)[]` format.

MetaCSP

{ readonly [resource: string]: string[] }

Type for a meta `Content-Security-Policy` tag in `{ resource: string[] }` format.

MetaData

Combined meta data for a website page.

PossibleMeta

Input metadata that can be parsed and converted to proper metadata.

joinMetaCSP

(csp: Nullish<MetaCSP>) => string | undefined

Turn a deconstructed CSP into a string.

joinTitles

(...titles: (string | undefined)[]) => string

Merge two page or site titles together, e.g. `Manchester Runners` + `Messages` becomes `Messages - Manchester Runners`

mergeMeta

(meta1: MetaData, meta2: PossibleMeta, caller: AnyCaller = mergeMeta) => MetaData

Merge two `MetaData` objects. - `title` is merged. - `URL` is resolved to an absolute URL, e.g. `./d/e/f` + `/a/b/c` becomes `https://d.com/a/b/c/d/e/f`

mergeMetaURL

(base: ImmutableURL | undefined, current: ImmutableURL | undefined, next: PossibleURL | undefined, params: PossibleURIParams | undefined, caller: AnyCaller = mergeMetaURL) => ImmutableURL | undefined

Merge two metadata URLs. - New URL is resolved relative to: current URL, new base URL, current base URL
