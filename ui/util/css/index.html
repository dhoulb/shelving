Classes

string | null | undefined | ClassesArray | ClassesDictionary

Set of classnames that can be joined. - `string` — used directly as a classname. - `null` or `undefined` — ignored. - Array of classnames — recursively parsed. - Dictionary of classnames — recursively parsed. - `true` or `false` item values in dictionaries use the string `key` of the item if the value is `true`, or ignore it if the value is `false`

ClassesArray

ClassesDictionary

CSSModule

ImmutableDictionary<string | undefined>

CSS modules mapping of local class names to hashed runtime class names.

getClass

(...classes: Classes[]) => string

Parse a list of possible `className` strings and join them into a single string. - See `Classes` type for parsing rules.

getModuleClass

(module: CSSModule, ...classes: Classes[]) => string

Parse a list of possible `className` strings, match them in a `CSSModule` dictionary, and join them into a single string. - See `Classes` type for parsing rules.
