Set of classnames that can be joined. - `string` — used directly as a classname. - `null` or `undefined` — ignored. - Array of classnames — recursively parsed. - Dictionary of classnames — recursively parsed. - `true` or `false` item values in dictionaries use the string `key` of the item if the value is `true`, or ignore it if the value is `false`
