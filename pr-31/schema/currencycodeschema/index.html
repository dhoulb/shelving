CurrencyCodeSchemaOptions

Options for a `CurrencyCodeSchema`

CurrencyCodeSchema

Type of `StringSchema` that defines a valid currency code.

CURRENCY_CODE

Valid currency code, e.g. `GBP`

NULLABLE_CURRENCY_CODE

Valid currency code, e.g. `GBP`, or `null`
