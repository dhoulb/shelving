refresh

(maxAge: number = ALWAYS_REFRESH) => Promise<boolean> | boolean

Fetch the result for this store now. - Triggered automatically when someone reads `value` or `loading`. - Refreshes are de-duplicated. Concurrent calls while a fetch is in-flight return the same promise. - Never throws — errors are stored as `reason`.

invalidate

() => void

Invalidate this store so a new fetch is triggered on the next read of `loading` or `value`. - Triggers `abort()` so any current awaits are cancelled.
