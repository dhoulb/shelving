IntrinsicMapping

{
	[K in keyof JSX.IntrinsicElements]?: ComponentType<JSX.IntrinsicElements[K]>;
}

Element mapping — maps intrinsic element type strings to React components. - Keys are element type names from `JSX.IntrinsicElements` (e.g. `"tree-file"`, `"tree-directory"`). - Each component must accept the props declared in `JSX.IntrinsicElements` for that element type.

Mapping

ImmutableDictionary<ComponentType>

A generic mapping object of any `string` to `ComponentType`

MappingProps

Props for the `Mapping` component returned by `createElementMapper()`.

MapperProps

Props for the `Mapper` component returned by `createElementMapper()`.

createMapper

(mapping: IntrinsicMapping = {}) => [Mapping: FunctionComponent<MappingProps>, Mapper: FunctionComponent<MapperProps>]

Create an element mapper — a `[Mapping, Mapper]` pair of React components. - `Mapping` — wraps a subtree to override or extend the element mapping for this mapper. - `Mapper` — maps children, replacing element types with registered components. Each mapper has its own React context, so different mappers (e.g. `TreeMenu`, `TreePage`) can have independent mappings without interfering with each other.
