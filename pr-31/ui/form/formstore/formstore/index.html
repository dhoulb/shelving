Store the current value of a form.

id

Unique ID for the form.

key

string

Key used for mounting the form

schema

DataSchema<T>

Schema for the current form.

messages

Store named error messages for individual fields. - Throwing a string triggers changes in this. - Rows prefixed with `fieldName:` are shown on those specific fields. - See `splitMessages()` in `shelving` for more information.

requireSchema

(name: K) => Schema<T[K]>

Get a named schema for a field of this form.

publish

(name: K, unsafeValue: T[K]) => void

Publish a value for a field of this form.

submit

(callback?: ((value: T, ...args: A) => void) | undefined, ...args: A) => boolean | Promise<boolean>

Validate and submit the current values of the form.
