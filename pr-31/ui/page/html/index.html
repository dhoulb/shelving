({ children }: HTMLProps) => ReactElement

Output a `<html>` element wrapping `<body id="root">`. - No `<head>` element is rendered. Head tags (`<title>`, `<meta>`, `<link>`, `<script>`) are emitted inline by `<Page>` / `<Head>` lower in the tree, and React 19 hoists them automatically — to the document `<head>` on the client, and to a generated `<head>` element during `renderToString` SSR. - This means the same component tree works for both modes without any shell-aware logic.
