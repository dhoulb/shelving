Create a View Transition for children of this component. - Allows known view transition types in `TransitionClasses` to be set to override. - These must correspond to a `::view-transition(.className)` that is set in CSS. - Supports variant classes, e.g. `<Transition overlay>` applies `::view-transition(.overlay)` from `Transition.css`
