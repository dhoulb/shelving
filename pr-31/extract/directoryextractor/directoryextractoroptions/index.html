Options for a directory extractor.

index

Matchables

Filenames to treat as the directory's index file. Matched case-insensitively.

extractors

ImmutableDictionary<FileExtractor>

Extractor dispatch table keyed by file extension (with leading dot, e.g. `".md"`). - Files with no matching extractor are silently skipped. - Defaults to `.md` (markdown), `.ts` (TypeScript), and `.txt` (plain text).

base

AbsolutePath

Absolute base path used to resolve relative paths passed to `extract()`.

ignore

Matchables

Glob patterns for entries to skip — applied to both files and directories. - Defaults to test and spec files: `["*.test.ts", "*.test.tsx", "*.spec.ts", "*.spec.tsx"]`. - Hidden entries (`.`-prefixed), underscore-prefixed entries, and `node_modules` are always skipped on top of these patterns.
