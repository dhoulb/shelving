Extractor

Base class for an extractor that converts input into a tree element. - Extractors are composable: outer extractors delegate to inner extractors. - The output type is always a `TreeElement` (or a more specific subtype like `TreeElement`).

mergeTreeElements

(primary: T, secondary: TreeElement) => T

Merge two file elements with the same `key`. - `title` and `path` are taken from `primary` (the higher-priority element). - `description` is taken from `primary` if set, otherwise from `secondary`. - `content` and `children` from both are concatenated (primary first).
