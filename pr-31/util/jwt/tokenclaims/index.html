iat

PossibleDate

"Issued at" date (defaults to "now"). - Not used for validation, but always set in the token payload. - Can be used to determine when the token was issued, and possibly revoke tokens issued before a certain date.

nbf

PossibleDate

"Not before" date. - When validating the token, tokens before this date will be rejected

exp

number

Expiry in milliseconds (defaults to "30 days"). - When validating the token, tokens after this date will be rejected
