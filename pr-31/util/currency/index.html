CurrencyCode

string

ISO 4217 currency code, e.g. `GBP` or `USD`.

CURRENCY_CODES

ImmutableArray<CurrencyCode>

Array of all supported currency codes in this runtime.

getCurrencyCode

(value: string) => CurrencyCode | undefined

Require that a value is a valid ISO 4217 currency code, and return it as a `Currency` type.

requireCurrencyCode

(value: string, caller: AnyCaller = requireCurrencyCode) => CurrencyCode

Require that a value is a valid ISO 4217 currency code, and return it as a `Currency` type.

getCurrencySymbol

(currency: CurrencyCode, caller: AnyCaller = getCurrencySymbol) => string

Get the display symbol used for a currency.

getCurrencyStep

(currency: CurrencyCode, caller: AnyCaller = getCurrencyStep) => number

Get the "step" value for a currency, i.e. the smallest fractional unit that is used for that currency. - E.g. `0.01` for USD, `0.001` for some cryptocurrencies, and `1` for JPY.
