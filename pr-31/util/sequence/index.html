Dispatch items in a sequence to a (possibly async) callback.

runSequence

(sequence: AsyncIterable<T, R | undefined, N | undefined>, onNext?: (value: T) => N | undefined, onError?: ErrorCallback, onReturn?: (value: R | undefined) => void) => (value?: R | undefined) => void

Iterate over a sequence until the returned `stop()` callback is called. Regarding errors: - Does not stop iterating on errors, simply sends the error to `onError()` and continues to iterate. - On the following iterator after throwing an error, a "generator" will return `done: true` (because they regard errors as concluding the iteration). - But the iterator protocol does not require this, so this continues to iterate until it's explicitly ended via the returned `stop()` callback.
