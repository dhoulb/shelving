(input: string, options: MarkupOptions, context = "block") => Elements

Parse a text string as Markdownish syntax and render it as elements. - Syntax is not defined by this code, but by the rules supplied to it.

Parameters

input: string
The string content possibly containing markup syntax, e.g. "This is a *bold* string.
options: MarkupOptions
An options object for the render.
context (optional)
The context to render in (defaults to `"block"`).

Returns

Elements
