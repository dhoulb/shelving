UNORDERED_REGEXP

UNORDERED_RULE

Unordered list. - Line starting with `-`, `*`, `•`, or `+` character followed by one or more space characters. - No spaces can appear before the bullet character. - Second-level list can be created by indenting with `\t` one tab. - List block is ended by `\n\n` two newline characters. - Sparse lists are not supported.
