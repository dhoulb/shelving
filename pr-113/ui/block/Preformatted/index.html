    Preformatted()

    ({ children, ...variants }: PreformattedProps) => ReactElement

    Preformatted block of text — rendered as <pre>. - Defaults to wrapping long lines (white-space: pre-wrap) so code fits the container width while preserving newlines and indentation within wrapped lines. - Pass nowrap to restore strict <pre> behaviour when exact whitespace matters (ASCII art, fixed-column tables).
