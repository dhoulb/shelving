URISchemaOptions

Allowed options for `URISchema`

URISchema

Type of `StringSchema` that defines a valid URI string. - Checks URI scheme against a whitelist (always). - URIs are limited to 512 characters, but generally these won't be data: URIs so this is a reasonable limit.

URI_SCHEMA

Valid URI string, e.g. `https://www.google.com`

NULLABLE_URI_SCHEMA

Valid URI string, e.g. `https://www.google.com`, or `null`
