UUIDSchema

Type of `StringSchema` that defines a valid UUID (versions 1-5). Defaults to any-version validation. - Input is trimmed and lowercased. - Falsy values are converted to empty string.

UUID

Any valid UUID (versions 1-5)

NULLABLE_UUID

Any valid UUID (versions 1-5) or null
