PhoneSchemaOptions

Options for a `PhoneSchema`

PhoneSchema

Type of `StringSchema` that defines a valid phone number. - Multiple string formats are automatically converted to E.164 format (starting with `+` plus). - Falsy values are converted to `""` empty string.

PHONE

Valid phone number, e.g. `+441234567890`

NULLABLE_PHONE

Valid phone number, e.g. `+441234567890`, or `null`
