CountrySchemaOptions

Allowed options for `CountrySchema`

CountrySchema

Schema that validates an ISO country code.

COUNTRY

Valid country code, e.g. `GB` (required because falsy values are invalid).

NULLABLE_COUNTRY

Valid country code, e.g. `GB`, or `null`
