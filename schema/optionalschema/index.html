OptionalSchemaOptions

OptionalSchema

Validate a property in an optional way, i.e. it can be the value, or `undefined` - If the prop is `undefined`, then `undefined` is returned. - When used with `validateData()` this means the prop can be silently skipped.

OPTIONAL

Make a property of a set of data optional, i.e. it can be the value or `undefined`
