KeySchema

Define a valid database key. - Characters that are not a-z, A-Z, 0-9 are removed. - Default minimum key length is 1 character. - Default maximum key length is 32 characters. - 32 characters is enough for UUIDs, as the 4 `-` hyphens are removed.

KEY

Valid database key.

NULLABLE_KEY

Valid optional database key.
