Validator

Object that can validate an unknown value with its `validate()` method.

ValidatorType

X extends Validator<infer Y> ? Y : never

Extract the type from a validator.

Validators

{ readonly [K in keyof T]: Validator<T[K]> }

A set of named validators in `{ name: Validator }` format.

ValidatorsType

{ readonly [K in keyof T]: ValidatorType<T[K]> }

Extract the type from a set of validators.

getValid

(value: unknown, validator: Validator<T>) => T | undefined

Require a valid value for a given validator, or return `undefined` if the value could not be validated.

requireValid

(value: unknown, validator: Validator<T>, caller: AnyCaller = requireValid) => T

Require a valid value for a given validator, or throw `RequiredError` if the value could not be validated.

validateItems

(unsafeItems: PossibleArray<unknown>, validator: Validator<T>) => Iterable<T>

Validate an iterable set of items with a validator.

validateArray

(unsafeArray: PossibleArray<unknown>, validator: Validator<T>) => ImmutableArray<T>

Validate an array of items.

validateDictionary

(unsafeDictionary: PossibleDictionary<unknown>, validator: Validator<T>) => ImmutableDictionary<T>

Validate the values of the entries in a dictionary object.

validateData

(unsafeData: Data, validators: Validators<T>) => T

Validate a data object with a set of validators. - Defined props in the object will be validated against the corresponding validator. - `undefined` props in the object will be set to the default value of that prop. - `undefined` props after validation will not be set in the output object.
