Represent a unit.

list

UnitList<K>

`UnitList` this unit belongs to.

key

K

String key for this unit, e.g. `kilometer`

options

Readonly<UnitFormatOptions> | undefined

Possible options for formatting these units.

to

(amount: number, targetKey?: K) => number

Convert an amount from this unit to another unit.

from

(amount: number, sourceKey?: K) => number

Convert an amount from another unit to this unit.

format

(amount: number, options?: UnitFormatOptions) => string

Format an amount with a given unit of measure, e.g. `12 kg` or `29.5 l` - Uses `Intl.NumberFormat` if this is a supported unit (so e.g. `ounce` is translated to e.g. `Unze` in German). - Polyfills unsupported units to use long/short form based on `options.unitDisplay`.
