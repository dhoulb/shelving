Turn ":year-:month" and `{ year: "2016"... }` etc into "2016-06..." etc.

Parameters

template: string
The template including template placeholders, e.g. `:name-${country}/{city}`
values: TemplateValues
An object containing values, e.g. `{ name: "Dave", country: "UK", city: "Manchester" }` (functions are called, everything else converted to string), or a function or string to use for all placeholders.
caller: AnyCaller (optional)

Returns

string
