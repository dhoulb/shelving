(data: Data, caller: AnyCaller = getXML) => string

Build an XML string from a data object. - Only data objects can be converted directly because XML requires named root and child elements. - `undefined` values are omitted from the output. - Nested data objects become nested XML elements.

Parameters

data: Data
The data object to serialize.
caller: AnyCaller (optional)
The calling function for error context.

Returns

string
