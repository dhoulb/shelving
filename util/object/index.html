DeepPartial

Deep partial object: deeply convert an object to its partial version. - Any value that extends `UnknownObject` has its props made partial. - Works deeply on nested objects too.

Deep mutable object: deeply convert an object to its mutable version. - Any value that extends `UnknownObject` has its props made mutable. - Works deeply on nested objects too.

DeepReadonly

Deep readonly object: deeply convert an object to its readonly version. - Any value that extends `UnknownObject` has its props made readonly. - Works deeply on nested objects too.
