(elements: Elements, match: (element: Element) => boolean, depth = Infinity) => Iterable<Element>

Filter elements using a match function. - Yields each matching `Element`, recursing into `props.children` up to `depth` levels.

Parameters

elements: Elements
The elements to filter.
match: (element: Element) => boolean
Function that tests whether an element should be yielded.
depth (optional)
Controls how many levels of children to recurse into (defaults to infinite depth).

Returns

Iterable<Element>
