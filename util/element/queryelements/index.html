(elements: Elements, query: Query<Element>, depth = Infinity) => Iterable<Element>

Query elements using a `Query<Element>` object, leveraging the same query system as `queryItems()`. - Extracts all elements via `getElements()` up to `depth` levels, then applies the query. - Supports filtering by any element property (e.g. `{ type: "tree-file" }`), sorting, and limiting.

Parameters

elements: Elements
The elements to query.
query: Query<Element>
A `Query<Element>` object (e.g. `{ type: "tree-file" }` or `{ type: ["tree-file", "tree-directory"] }`).
depth (optional)
Controls how many levels of children to recurse into (defaults to infinite depth).

Returns

Iterable<Element>
