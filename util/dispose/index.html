DisposableMap

Version of `Map` that has disposable values. - Old values are disposed when they're set to a new value. - Values are disposed when they're deleted from the map. - Values are disposed when all items are cleared. - All items are cleared and their values are disposed when this map itself is disposed.

DisposableSet

Version of `Set` that has disposable items. - Values are disposed when they're deleted from the map. - Values are disposed when all items are cleared. - All items are cleared (and disposed) when this map itself is disposed.
