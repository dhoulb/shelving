(left: L, right: R, recursor?: MergeRecursor) => L & R

Merge two data objects. Properties that exist in `right` will replace or be deeply merged with properties in `left`. - Only works on enumerable own keys (as returned by `Object.keys()`). - Always returns a new object (or the `left` object if no changes were made). - Resulting object is `cleaned`, i.e. properties in `left` or `right` that are `undefined` will be removed from the merged object.

Parameters

left: L
right: R
recursor: MergeRecursor (optional)
Function that merges each property of the object.

Returns

L & R
