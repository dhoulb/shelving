(num: number, precision = 0) => number

Round a number to a specified set of decimal places. - Better than `Math.round()` because it allows a `precision` argument. - Better than `num.toFixed()` because it trims excess `0` zeroes.

Parameters

num: number
The number to round.
precision (optional)
Maximum number of digits shown after the decimal point (defaults to 10).

Returns

number
