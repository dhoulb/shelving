PossibleNumber

number | string | Date

Values that can be converted to a number.

isNumber

(value: unknown, min = Number.NEGATIVE_INFINITY, max = Number.POSITIVE_INFINITY) => value is number

Is a value a finite number?

assertNumber

(value: unknown, min?: number, max?: number, caller: AnyCaller = assertNumber) => asserts value is number

Assert that a value is a finite number.

getNumber

(value: unknown) => number | undefined

Convert an unknown value to a finite number, or return `undefined` if it cannot be converted. - Note: numbers can be non-finite numbers like `NaN` or `Infinity`. These are detected and will always return `undefined` Conversion rules: - Finite numbers return numbers. - `-0` is normalised to `0` - Strings are parsed as numbers using `Number.parseFloat()` after removing all non-numeric characters. - Dates return their milliseconds (e.g. `date.getTime()`). - Everything else returns `undefined`

requireNumber

(value: PossibleNumber, min?: number, max?: number, caller?: AnyCaller) => number

Convert a possible number to a finite number, or throw `ValueError` if the value cannot be converted.

isInteger

(value: unknown, min = Number.MIN_SAFE_INTEGER, max = Number.MAX_SAFE_INTEGER) => value is number

Is an unknown value an integer (optionally with specified min/max values).

assertInteger

(value: unknown, min?: number, max?: number, caller: AnyCaller = assertInteger) => asserts value is number

Assert that a value is an integer.

getInteger

(value: unknown) => number | undefined

Convert an unknown value to an integer, or return `undefined` if it cannot be converted. Conversion rules: - Integers return integers. - `-0` is normalised to `0` - Strings are parsed as integers using `parseInt()` after removing non-numeric characters. - Dates return their milliseconds (e.g. `date.getTime()`). - Everything else returns `undefined`

requireInteger

(value: PossibleNumber, min?: number, max?: number, caller: AnyCaller = requireInteger) => number

Convert a possible number to an integer, or throw `ValueError` if the value cannot be converted.

isBetween

(num: number, min: number, max: number) => boolean

Is a number within a specified range?

roundStep

(num: number, step = 1) => number

Round numbers to a given step.

roundNumber

(num: number, precision = 0) => number

Round a number to a specified set of decimal places. - Better than `Math.round()` because it allows a `precision` argument. - Better than `num.toFixed()` because it trims excess `0` zeroes.

truncateNumber

(num: number, precision = 0) => number

Truncate a number to a specified set of decimal places. - Better than `Math.trunc()` because it allows a `precision` argument.

boundNumber

(num: number, min: number, max: number) => number

Bound a number between two values. - e.g. `12` bounded by `2` and `8` is `8`

wrapNumber

(num: number, min: number, max: number) => number

Wrap a number between two values. - Numbers wrap around between min and max (like a clock). - e.g. `12` bounded by `2` and `8` is `6` - Words in both directions. - e.g. `-2` bounded by `2` and `8` is `4`

getPercent

(numerator: number, denumerator = 100) => void

Get a number as a percentage of another number.

sumNumbers

(nums: Iterable<number>) => number

Sum an iterable set of numbers and return the total.

getClosestNumber

(nums: Iterable<T>, target: number) => T | undefined

Find the number that's closest to a target in an iterable set of numbers.
