({ headers: aHeaders, signal: aSignal, ...a }: RequestOptions = {}, { headers: bHeaders, signal: bSignal, ...b }: RequestOptions = {}) => RequestOptions

Merge provider-level and call-level request options. - Scalar options from `b` override `a`. - Header dictionaries are merged so call-level headers override default headers by key. - Abort signals are merged, so either abort signal will cancel the request.

Parameters

{ headers: aHeaders, signal: aSignal, ...a }: RequestOptions (optional)
{ headers: bHeaders, signal: bSignal, ...b }: RequestOptions (optional)

Returns

RequestOptions
