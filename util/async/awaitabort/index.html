(signal: AbortSignal) => Promise<never>

Get a promise that rejects with the signal's reason when an `AbortSignal` fires. - Rejects immediately if the signal is already aborted. - Use with `awaitRace()` to cancel a concurrent operation when a signal fires.

signal: AbortSignal

Promise<never>
