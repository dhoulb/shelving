Transforms

{
	readonly [K in keyof I]?: (input: I[K], ...args: A) => O[K];
}

Set of named transforms for a data object (or `undefined` to skip the transform).

mapItems

(items: Iterable<I>, transform: (v: I, ...args: A) => O, ...args: A) => Iterable<O>

Modify a set of items using a transform.

mapArray

(arr: Iterable<I>, transform: (v: I, ...args: A) => O, ...args: A) => ImmutableArray<O>

Modify the items of an array using a transform.

mapProps

(obj: I, transform: (prop: Prop<I>, ...args: A) => Value<O>, ...args: A) => O

Modify the _values_ of the props of an object using a transform.

mapDictionary

(dictionary: ImmutableDictionary<I>, transform: (item: DictionaryItem<I>, ...args: A) => O, ...args: A) => ImmutableDictionary<O>

Modify the _values_ of a dictionary using a transform.

mapEntries

(entries: Iterable<Entry<K, I>>, transform: (entry: Entry<K, I>, ...args: A) => O, ...args: A) => Iterable<Entry<K, O>>

Modify the _values_ of a set of entries using a transform.

transformObject

(obj: T, transforms: Transforms<T, T | Partial<T>, A>, ...args: A) => T

Transform an object using a set of named transforms.

mapSequence

(sequence: AsyncIterable<I>, transform: (input: I, ...args: A) => O | PromiseLike<O>, ...args: A) => AsyncIterable<O>

Transform items in a sequence as they are yielded using a (potentially async) transform.
