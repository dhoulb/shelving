Timeout

Create a new Timeout. Wrapper for `setTimeout()` that... - Keeps track of the reference returned from `setTimeout()` - Clears the any existing timeout when a new timeout is set. - Allows a default delay to be set that is applied to all new timeouts that don't have a delay set.
