Bytes

Uint8Array<ArrayBuffer>

We store a sets of bytes as a `Uint8Array` byte sequence.

PossibleBytes

Bytes | ArrayBuffer | string

Types that can be converted to a `Uint8Array` byte sequence.

isBytes

(value: unknown) => value is Bytes

Is an unknown value a set of bytes?

assertBytes

(value: unknown, min = 0, max = Number.POSITIVE_INFINITY, caller: AnyCaller = assertBytes) => asserts value is Bytes

Assert that an unknown value is a `Uint8Array` byte sequence.

getBytes

(value: unknown) => Uint8Array<ArrayBuffer> | undefined

Convert an unknown value to a `Uint8Array<ArrayBuffer>` byte sequence, or `undefined` if the value cannot be converted. - `Uint8Array` values are returned as-is (if backed by an `ArrayBuffer`) or copied to a new `Uint8Array` (if not). - `ArrayBuffer` instances are converted to `Uint8Array` - Strings are encoded as UTF-8 characters in a `Uint8Array` - Everything else returns `undefined`

requireBytes

(value: PossibleBytes, min = 0, max = Number.POSITIVE_INFINITY, caller: AnyCaller = requireBytes) => Uint8Array<ArrayBuffer>

Convert a possible set of bytes to a `Uint8Array` byte sequence, or throw `RequiredError` if the value cannot be converted.
