Reuseable utilities for dealing with filesystem paths.

AbsolutePath

`/` | `/${string}`

Absolute path string starts with `/` slash.

RelativePath

`.` | `./` | `./${string}`

Relative path string is `.` dot, or starts with `./` dot slash.

Path

AbsolutePath | RelativePath

Either an absolute or relative path.

PossiblePath

string

Things that can be converted to a path.

isAbsolutePath

(path: string) => path is AbsolutePath

Is a string path an absolute path?

isRelativePath

(path: string) => path is RelativePath

Is a string path an relative path?

getPath

(inputPath: Nullish<PossiblePath>, inputBase: AbsolutePath = "/") => AbsolutePath | undefined

Resolve a relative or absolute path and return the absolute path, or `undefined` if not a valid path. - Normalises runs of `//` more than one slash. - Normalises `\` windows paths. - Strips trailing slashes.

cleanPath

(path: AbsolutePath) => AbsolutePath

Remove trailing slashes and normalise `//` runs in a path.

cleanPath

(path: RelativePath) => RelativePath

cleanPath

(path: Path) => Path

cleanPath

(path: string) => string

cleanPath

(path: string) => string

requirePath

(path: PossiblePath, base?: AbsolutePath, caller: AnyCaller = requirePath) => AbsolutePath

Resolve a relative or absolute path and return the path, or throw `RequiredError` if not a valid path. - Internally uses `new URL` to do path processing but shouldn't ever reveal that fact. - Returned paths are cleaned with `cleanPath()` so runs of slashes and trailing slashes are removed.

matchPathPrefix

(target: PossiblePath, base: PossiblePath, caller: AnyCaller = matchPathPrefix) => AbsolutePath | undefined

Match and strip a base path prefix from a path using segment-aware pathname rules. - Both inputs must be absolute paths that begin with `/`. - Returns `/` when the paths are an exact match.

isPathActive

(target: AbsolutePath, current: AbsolutePath) => boolean

Is a target path active?

isPathProud

(target: AbsolutePath, current: AbsolutePath) => boolean

Is a target path proud (i.e. is the current path, or is a child of the current path)?

splitAbsolutePath

(path: AbsolutePath | Segments) => Segments

Get the "segments" in an absolute path.

joinAbsolutePath

(path: Segments | AbsolutePath) => AbsolutePath

Join a set of path segments to form an absolute path.
