(path: PossiblePath, base?: AbsolutePath, caller: AnyCaller = requirePath) => AbsolutePath

Resolve a relative or absolute path and return the path, or throw `RequiredError` if not a valid path. - Internally uses `new URL` to do path processing but shouldn't ever reveal that fact. - Returned paths are cleaned with `cleanPath()` so runs of slashes and trailing slashes are removed.

Parameters

path: PossiblePath
Absolute path e.g. `/a/b/c`, relative path e.g. `./a` or `b` or `../c`, URL string e.g. `http://shax.com/a/b/c`, or `URL` instance.
base: AbsolutePath (optional)
Absolute path used for resolving relative paths in `possible`
caller: AnyCaller (optional)

Returns

AbsolutePath
