API provider that always sends request bodies as JSON and parses responses as JSON.

parseResponse

(_endpoint: Endpoint<PP, RR>, response: Response, caller: AnyCaller = this.parseResponse) => Promise<RR>

Parse a JSON `Response` for an endpoint. - Non-2xx responses become `ResponseError`. - JSON is parsed even if the server omitted or mis-set the response content type.
