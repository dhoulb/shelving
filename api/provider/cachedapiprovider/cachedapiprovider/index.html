API provider wrapper that serves requests through an `APICache`. - Constructor accepts a `source` provider and an optional default `maxAge`. - On `call(...)`, triggers `cache.refresh(maxAge)` for the endpoint+payload before awaiting `cache.call(...)`. - `invalidate`, `invalidateAll`, `refresh`, and `refreshAll` pass through to the underlying cache and use `this.maxAge` as the default refresh timing.
