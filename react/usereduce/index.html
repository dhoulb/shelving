useReduce

(reduce: (previous: T | undefined, ...a: A) => T, ...args: A) => T

Use memoised value with reduction logic. - Calls `reduce()` on each render. - First argument `previous` is either `undefined` (on first render) or the value returned by `reduce()` on the previous render (on subsequent renders). - `reduce()` can implement custom logic to decide whether to return the previous value or a new one. - Returns whatever `reduce()` returns on this render.
