DBContext

createDBContext

(provider: DBProvider<I, T>) => DBContext<I, T>

Create a data context - Allows React elements to call `useItem()` and `useQuery()` to access items/queries in a database provider. - If the database has a `CacheDBProvider` in its chain then in-memory data will be used in the returned stores.
