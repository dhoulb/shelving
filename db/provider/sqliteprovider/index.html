SQLiteProvider

Abstract SQLite provider with JSON1 function support for nested keys, array containment, and array mutations. Note the following compatibility caveats: - For `with` and `omit` updates this does not preserve ordering of the original array. - For `with` and `omit` updates this does not guarantee equality for de-duplication when working with nested objects or arrays.
