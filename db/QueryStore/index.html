    QueryStoreclass

    extends FetchStore<Items<I, T>>implements Iterable<Item<I, T>>
    new QueryStore<I, T>(collection: Collection<string, I, T>, query: Query<Item<I, T>>, provider: DBProvider<I>, memory?: MemoryDBProvider<I>)
    ParamTypeDefault
    collectionCollection<string, I, T>-The collection the query runs against.
    queryQuery<Item<I, T>>-The query that selects the items to track.
    providerDBProvider<I>-The database provider to fetch the items from.
    memoryMemoryDBProvider<I>-Optional memory provider used to seed the initial value and drive realtime updates.
    Return
    QueryStore<I, T>Store that runs a query against a collection from a database provider and tracks its matching items.

    Store that runs a query against a collection from a database provider and tracks its matching items.

    • Holds an Items value (the array of matching items) and is iterable over those items.
    • Seeds from a MemoryDBProvider snapshot when available, and subscribes to realtime updates.

    Examples

    const store = new QueryStore(collection, query, provider);
 for (const item of store) console.log(item);
