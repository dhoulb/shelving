    • Tag()

    (props: TagProps) => ReactElement

    Small inline label used to annotate other content.

    • Delegates to getClickable() — renders as <a> when href is set, otherwise <button>.
    • Accepts a status variant (success, info, error, etc.) or a raw colour variant (red, blue, purple, etc.).

    Parameters

    props: TagProps

    Returns

    ReactElement

    Examples

    <Tag success>Active</Tag>
    <Tag purple href="/beta">Beta</Tag>
