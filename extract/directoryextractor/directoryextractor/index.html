Extractor that walks a directory on disk and produces a `DirectoryElement` tree. - Recursively descends into subdirectories. - Detects an index file (e.g. `README.md`) and absorbs its content/description/title as the directory's own. - Dispatches non-index files to a matching `FileExtractor` based on extension; files with no matching extractor are silently skipped. - Merges file elements that share the same `key` (e.g. `TEMPLATE.md` and `template.ts`) using each extractor's `priority`. - Throws if a directory and a file (or two directories) share the same `key`.
