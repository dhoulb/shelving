TypescriptExtractor

File extractor that parses a TypeScript source file into a tree element. - Uses the TypeScript compiler API to parse the AST. - Extracts exported, public, non-`_`-prefixed declarations as `tree-documentation` children. - Top-of-file JSDoc comment becomes the file's `content`. - Does not set `title` — TS source files have no confident title source. The renderer falls back to `name`.
