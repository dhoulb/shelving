    Navigation()

    ({ children, ...meta }: NavigationProps) => ReactElement

    Top-level navigation provider. - Owns a single NavigationStore initialised from the surrounding <Meta> url/base. - Intercepts same-origin anchor clicks (excluding download anchors) and turns them into forward() calls. - Listens for popstate to sync the store with browser back/forward. - Publishes the live URL via <Meta url={…} params={…}> so descendant <Router>s re-render on navigation.

    NavigationIsolate()

    ({ children }: NavigationIsolateProps) => ReactElement

    Force a full remount of children whenever the navigation URL changes. - Use to reset stateful sub-trees on navigation.
