    Variants

    Variants list is a dictionary of booleans. - true or false item values in dictionaries use the string key of the item if the value is true, or ignore it if the value is falsy. - Anything that is not true has no effect, so other values can be passed in and will be ignored. This means you can pass the entire props into this and it'll work just fine.
