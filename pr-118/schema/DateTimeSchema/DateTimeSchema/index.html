    • DateTimeSchema

    Define a valid UTC date in ISO 8601 format, e.g. 2005-09-12T18:15:00.000Z

    • The date includes the Z suffix to indicate UTC time, this ensures consistent transfer of the date between client and server.
    • If you wish to define an abstract date without a timezone, e.g. a birthday or anniversary, use DateSchema instead.
    • If you wish to define an abstract time without a timezone, e.g. a daily alarm, use TimeSchema instead.
