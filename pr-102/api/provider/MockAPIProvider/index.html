    MockAPIProvider

    Provider that logs API calls without sending network requests. - Extends ThroughAPIProvider to delegate request building and response parsing to a source APIProvider. - The source provider's fetch() is never called — this provider intercepts all fetches and routes them through a RequestHandler.
