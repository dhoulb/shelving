    • util/entity

    An entity is a compact "type:id" string that encodes both the kind of resource and its identifier in one value, e.g. "challenge:a1b2c3". These helpers parse and validate those strings so callers never need to split on : themselves.

    Usage

    Splitting an entity string

    Using the Entity<T> type

    Go

    getEntity()

    (entity: Entity<T>) => [type: T, id: string]
    (entity: Nullish<Entity<T>>) => [type: T, id: string] | EmptyEntity
    (entity: Nullish<string>) => [type: string, id: string] | EmptyEntity
    (entity: Nullish<string>) => [type: string, id: string] | EmptyEntity

    Split an optional entity tag like challenge:a1b2c3 into ["challenge", "a1b2c3"], or return EmptyEntity if the entity was invalid.

    Go

    requireEntity()

    (entity: Entity<T>, caller?: AnyCaller) => [type: T, id: string]
    (entity: string, caller?: AnyCaller) => [type: string, id: string]
    (entity: string, caller: AnyCaller = requireEntity) => [type: string, id: string]

    Split an entity tag like challenge:a1b2c3 into type and id, or throw RequiredError if the entity tag was invalid.
